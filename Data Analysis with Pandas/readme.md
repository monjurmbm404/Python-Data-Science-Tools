# 🐼 Python Data Science Tools

## 📊 Data Analysis with Pandas (Beginner → Advanced)

This repository is a complete **Pandas learning journey** designed for anyone who wants to master **Data Analysis with Python** from scratch to advanced level.

It follows a **structured Day-by-Day roadmap (33 days)** covering everything from basic data handling to real-world data projects.

---

## 🚀 What You Will Learn

- Pandas fundamentals (Series, DataFrame)
- Data loading & inspection
- Data cleaning & preprocessing
- Filtering & transformation
- Grouping & aggregation
- Merging & joining datasets
- Time series analysis
- Advanced data manipulation
- Real-world project experience
