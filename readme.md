## 🎯 Overview

This repository provides a complete structured learning journey for Python Data Science covering:

- Data Analysis
- Data Visualization
- Statistical Thinking
- Interactive Dashboards
- Real-world Project Development

It is designed with a **day-by-day roadmap approach** to ensure consistent learning and skill progression.

---

## 🗺️ Learning Roadmap Overview

| Module        | Focus Area                | Duration | Level                   |
| ------------- | ------------------------- | -------- | ----------------------- |
| 🧮 NumPy      | Numerical Computing       | 18 Days  | Beginner → Intermediate |
| 🐼 Pandas     | Data Analysis             | 33 Days  | Beginner → Advanced     |
| 📊 Matplotlib | Static Visualization      | 35 Days  | Beginner → Advanced     |
| 📊 Seaborn    | Statistical Visualization | 35 Days  | Intermediate → Advanced |
| 📊 Plotly     | Interactive Dashboards    | 40 Days  | Advanced                |

## 🧭 Learning Path Recommendation

Follow this order for best results:

1. 🧮 NumPy → Build numerical foundation
2. 🐼 Pandas → Data manipulation & analysis
3. 📊 Matplotlib → Core visualization skills
4. 📊 Seaborn → Statistical visualization & EDA
5. 📊 Plotly → Interactive dashboards & advanced analytics

## ⚙️ Setup & Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/monjurmbm404/Python-Data-Science-Tools.git
cd Python-Data-Science-Tools
```

### 2️⃣ Install Dependencies

```bash
pip install numpy pandas matplotlib seaborn plotly
```

### 3️⃣ Start Learning

- Open any module folder
- Follow the roadmap
- Practice with code
